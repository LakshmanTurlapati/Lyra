# Lyra Training Dataset

## Dataset Description

Synthetic instruction-tuning dataset for fine-tuning SmolLM2-1.7B on tool calling, code generation, and general knowledge tasks. All training data is generated by Claude Opus and processed through a local quality pipeline with deduplication, format validation, tokenizer validation, and heuristic quality checks.

The dataset uses TRL-native `messages`/`role`/`content` records with SmolLM2-specific `<tool_call>` XML wrappers for tool-call examples. It is validated against the SmolLM2-1.7B tokenizer and chat template before inclusion.

## Creation Methodology

1. Template-based generation across tool-calling, code, and knowledge domains.
2. Batch generation using Claude Opus prompts and local JSONL outputs.
3. Quality curation with n-gram Jaccard deduplication, Pydantic format validation, and heuristic scoring.
4. Tokenizer validation against SmolLM2-1.7B chat template constraints.
5. HuggingFace `DatasetDict` assembly with train, validation, and test splits.

## Dataset Statistics

Current `datasets/assembled/` split sizes:

| Split | Samples | Tool-Calling | Code | Knowledge |
|-------|---------|--------------|------|-----------|
| Train | 11,963 | 10,793 | 584 | 586 |
| Validation | 666 | 600 | 33 | 33 |
| Test | 663 | 599 | 32 | 32 |
| **Total** | **13,292** | **11,992** | **649** | **651** |

Domain distribution is intentionally tool-call heavy in the current assembled dataset: about 90.2% tool-calling, 4.9% code, and 4.9% knowledge.

## Limitations

- **Synthetic data**: All samples are generated by Claude Opus and may not capture the full distribution of natural user conversations.
- **Format-specific**: Tool calls use SmolLM2-specific XML-wrapped JSON, not OpenAI function calling format.
- **English only**: No multilingual samples are included.
- **Domain imbalance**: The current assembled dataset is heavily weighted toward tool-calling to support regression investigation.
- **Bounded coverage**: Code and knowledge samples come from finite template and topic pools, so breadth is intentionally limited.
