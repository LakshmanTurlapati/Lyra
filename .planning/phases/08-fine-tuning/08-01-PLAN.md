## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI args -> training config | User-provided paths and numeric values enter the training loop |
| HuggingFace Hub -> local model | Pre-trained model weights downloaded from external source |
| Dataset files -> training data | Arrow files loaded from disk into training pipeline |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Tampering | Model loading | accept | Use safetensors format only (transformers default). Trust HuggingFace Hub models per CLAUDE.md constraint on SmolLM2-1.7B-Instruct. |
| T-08-02 | Tampering | CLI path arguments | mitigate | Validate output_dir and merged_dir via argparse; use Path objects; do not construct paths from untrusted concatenation. Resolve relative to project root. |
| T-08-03 | Information Disclosure | wandb API key | mitigate | wandb uses WANDB_API_KEY env var (standard pattern). Script never logs or prints API keys. Only enabled via explicit --wandb flag per D-11. |
| T-08-04 | Denial of Service | Memory exhaustion on MPS | accept | Document memory requirements in script help text. batch_size=4 + grad_accum=4 fits comfortably in 36GB unified memory. User can reduce batch size if needed. |
</threat_model>
