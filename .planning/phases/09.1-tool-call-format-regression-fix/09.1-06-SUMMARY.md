# Plan 09.1-06 Summary: Retrained Lyra Evaluation

## Status

Blocked at human verification checkpoint.

The retrained Lyra v2 model was evaluated sequentially on the same Phase 9 benchmark set. All planned result artifacts were produced and merged, but the success criteria did not pass:

- D-09 failed: tool-call-format scored 0.0634, below the >0.41 target and below the base model score of 0.4065.
- D-10 partially failed: MMLU degraded 7.5% from base, exceeding the allowed 5% degradation guard.
- ARC-Challenge and HellaSwag passed the D-10 guard.

Custom eval:


Merge:

```bash
.venv/bin/python -m scripts.eval_merge \
  --first results/lyra_v2_knowledge.json \

  --output results/lyra_v2_full.json
```

Compare:

```bash
.venv/bin/python -m scripts.eval_compare \
  --baseline results/base_full.json \
```
