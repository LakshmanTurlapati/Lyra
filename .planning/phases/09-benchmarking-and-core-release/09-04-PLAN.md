STRICTLY SEQUENTIAL EXECUTION — DO NOT PARALLELIZE ANY EVAL COMMANDS.

consumed 80GB+ RAM and crashed the 64GB machine. Every eval command MUST complete and its

Rules:
1. Never use parallel Bash tool calls for any command that loads a model
2. Never use run_in_background for model evaluation commands

4. Between model-loading commands, the previous Python process must fully exit to free MPS memory
</memory_constraint>
