## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JSONL file -> json.loads | Untrusted JSON content from curated files loaded into memory |
| Dataset -> validate_format | Assembled data validated against Pydantic schema |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-01 | Tampering | JSONL loading (load_domain_jsonl) | mitigate | Line-by-line json.loads with try/except per line; skip malformed lines with warning log. No eval/exec. Pydantic Conversation model validates structure. |
| T-07-02 | Information Disclosure | _quality metadata leak | mitigate | Explicit pop("_quality", None) on every sample during loading; test verifies no _quality in output. |
| T-07-03 | Tampering | Path traversal in file paths | accept | All paths are hardcoded constants in DOMAIN_SOURCES dict (no user-supplied paths in assembly logic). Output dir from CLI is a local path validated by pathlib. |
| T-07-04 | Denial of Service | Memory exhaustion on large files | accept | Dataset is fixed at ~3,630 samples (~10-20MB). Not a concern at this scale. |
</threat_model>
