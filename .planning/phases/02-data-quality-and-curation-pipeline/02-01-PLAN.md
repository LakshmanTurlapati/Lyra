       - Turn length balance: max(assistant_lengths) / max(1, min(user_lengths)) > config.get("max_turn_ratio", 50)
       Returns {"signal": "naturalness", "score": max(0.0, 1.0 - len(issues)*0.3), "pass": len(issues)==0, "issues": list}

    4. score_sample(sample: dict, config: dict) -> dict: Runs all 4 signals (format, completeness, naturalness, diversity placeholder). Diversity signal always returns pass=True/score=1.0 in this module (actual diversity is handled by dedup.py at batch level per D-03 "Content diversity: near-duplicate detection within batch"). Returns:
       {"pass": all signals pass, "score": min(all signal scores), "signals": {"format": {...}, "completeness": {...}, "naturalness": {...}, "diversity": {...}}}

  <action>
    Create scripts/dedup.py implementing per D-04 and D-05:

    1. extract_ngrams(text: str, n: int = 3) -> set[str]: Normalize text (lowercase, collapse whitespace with " ".join(text.lower().split())). If len(normalized) < n, return {normalized} if non-empty else empty set. Otherwise return {text[i:i+n] for i in range(len(text) - n + 1)}.

    2. jaccard_similarity(set_a: set, set_b: set) -> float: If both empty, return 1.0. Otherwise return len(intersection) / len(union).

    3. get_dedup_text(sample: dict, scope: str = "response") -> str: Extract comparison text based on scope:
       - "response": join all assistant messages' content (skip None content from tool_call messages)
       - "prompt": join all user messages' content
       - "full": join all messages' content (skip None)
       Messages accessed via sample["messages"], filtering by role.

    4. deduplicate_batch(samples: list[dict], config: dict) -> list[dict]: Read config keys: ngram_size (default 3), dedup_threshold (default 0.7), dedup_scope (default "response"). Pre-compute n-gram sets for all samples. O(n^2) all-pairs comparison. For each pair (i, j) where i < j, if jaccard_similarity >= threshold, mark j for removal. Return samples where keep[i] is True. Uses NO external dependencies per D-04.
